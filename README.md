# 🎓 Student Management System

A web-based CRUD application to manage student records using Django framework.

## ✨ Features
- Add, update, delete, and view students
- Responsive UI with Bootstrap
- SQLite database

## 🔧 Tech Stack
- Django (Python)
- SQLite
- HTML, CSS, Bootstrap
